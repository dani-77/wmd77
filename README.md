# wmd77

This is my Penrose based X11 Tiling Window Manager.

## Dependencies

- cargo & rust

- libX11-devel, libXft-devel

- make

### Optional dependencies

- feh

- xautolock

- slock

- xcompmgr

- udiskie

# Install

To install just:

```
make && sudo make install
```
